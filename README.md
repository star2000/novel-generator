# 高质量长篇小说生成器

一个基于AI的小说自动生成工具，能够根据用户输入的要求，生成完整的小说内容，包括总纲、设定集、卷大纲、章节大纲和章节正文。

## 功能特点

- 自动生成小说书名
- 生成小说总纲和设定集
- 按照总纲自动划分卷和章节
- 为每个章节生成大纲和正文
- 支持对生成的小说进行评价

## 环境安装步骤

1. [安装uv包管理器](https://docs.astral.sh/uv/getting-started/installation/) 和 [Ollama 本地ai启动器](https://ollama.com/download)
2. 克隆项目到本地：
   ```sh
   git clone https://github.com/star2000/novel-generator.git
   ```
3. 进入项目目录：
   ```sh
   cd novel-generator
   ```
4. 下载模型调参待用：
   ```sh
   ollama pull qwen3.5:4b
   ```

## 基本用法

### 生成小说

**novel_generator.py**：

- 核心的小说生成逻辑
- 实现了从用户输入到完整小说的生成流程
- 包括书名生成、总纲生成、设定集生成、卷和章节的生成

**参数说明：**

- `--model, -m`：模型名称，默认为"qw"
- `--user-input, -i`：小说生成要求，描述你想要的小说类型、风格、主题等
- `--book-name, -n`：小说书名，如不提供则自动生成
- `--output-dir, -o`：输出目录路径，默认为"./dist/"

**示例：**

```sh
# 无参数，询问用户输入要求
uv run novel_generator.py

# 仅输入要求，参数传递版
uv run novel_generator.py --user-input "一部关于未来世界的科幻小说，主角是一名人工智能科学家"

# 仅书名，用于中断后继续生成
uv run novel_generator.py --book-name "剑影江湖"

# 全参数自定义
uv run novel_generator.py --model "qw" --user-input "一部关于未来世界的科幻小说，主角是一名人工智能科学家" --book-name "剑影江湖" --output-dir "./dist/"
```

### 评价小说

**novel_review.py**：

- 实现小说评价功能
- 分析小说内容并生成评价

**参数说明：**

- `--model, -m`：模型名称，默认为"qw"
- `--book-name, -n`：小说书名，如不提供则评价输出目录下所有小说
- `--output-dir, -o`：输出目录路径，默认为"./dist/"

**示例：**

```sh
# 无参数，评价已生成的所有小说
uv run novel_review.py

# 仅指定小说书名，评价该小说
uv run novel_review.py --book-name "剑影江湖"

# 全参数自定义
uv run novel_review.py --model "qw" --book-name "剑影江湖" --output-dir "./dist/"
```

## 工作流程

1. **输入处理**：获取用户输入的小说要求
2. **书名生成**：根据用户要求生成合适的书名
3. **总纲生成**：生成小说的总纲，包括分卷情况
4. **设定集生成**：生成小说的详细设定，包括世界背景、角色、地点等
5. **分卷处理**：根据总纲生成各卷的大纲
6. **章节处理**：为每卷生成章节大纲和正文
7. **内容检查**：检查生成的内容是否符合设定和逻辑
8. **输出保存**：将生成的内容保存到指定目录

## 产物结构

生成的小说会按照以下结构组织：

```
<输出目录>/
└── <小说书名>/
    ├── 要求.md         # 用户输入的小说要求
    ├── 总纲.md         # 小说总纲，包括各卷的大致内容
    ├── 设定集.md       # 小说设定，包括世界、角色、地点等
    ├── 第1卷-<卷名>/
    │   ├── 大纲.md     # 卷大纲，包括该卷各章节的大致内容
    │   ├── 第1章-<章节名>/
    │   │   ├── 大纲.md # 章节大纲
    │   │   ├── 正文.diff # 正文清理前后差异
    │   │   ├── 正文.md # 章节正文
    │   │   └── 正文.txt # 清理后的章节正文
    │   ├── 第2章-<章节名>/
    │   │   ├── 大纲.md
    │   │   ├── 正文.diff
    │   │   ├── 正文.md
    │   │   └── 正文.txt
    │   └── ...
    ├── 第2卷-<卷名>/
    │   ├── 大纲.md
    │   ├── 第1章-<章节名>/
    │   │   ├── 大纲.md
    │   │   ├── 正文.diff
    │   │   ├── 正文.md
    │   │   └── 正文.txt
    │   └── ...
    └── ...
```
